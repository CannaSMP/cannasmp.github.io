Main Server Theme
-----------------
CannaSMP is themed around cannabis/weed culture, but the server should still
look clean, polished, and readable. The visual identity uses bright green,
dark green, black/dark charcoal, and white highlights.

The server has mature-theme notices because players may see swearing,
cannabis references, smoking references, tobacco references, and other mature
jokes.

Core Player Systems
-------------------
Spawn:
- Normal players are put in adventure mode.
- PvP is disabled.
- Spawn is protected/unbreakable.
- Dropping items is blocked where configured.

KOTH:
- KOTH world exists and has its own scoreboard/filter setup.
- KOTH should allow PvP except in the KOTH spawn safe zone.
- KOTH has a worldborder/containment setup so players cannot escape the arena.
- KOTH sidebar shows status, holder, time, captured time, reward, ping, and TPS.

AFK World:
- AFK world is named afk.
- /afk teleports players to the AFK world after a 5 second delay.
- Moving cancels the /afk teleport.
- AFK teleport location:
  /execute in minecraft:afk run tp @s 0.50 -54.00 0.50 180.00 0.00
- Players in AFK world are in adventure mode.
- PvP is disabled like spawn.
- Shards are only earned in the AFK world.
- 1 shard = 1 minute in the AFK zone.
- Essentials /afk is disabled/replaced.
- The AFK tag should only apply when a player is in the AFK world.

Economy And Currencies
----------------------
Money:
- Uses Vault-compatible economy.
- Balance appears in scoreboard and menus.
- Commands include /bal, /balance, and /pay.

AFK Shards:
- Earned only in the AFK world.
- Used in the shard shop.
- Has placeholders for scoreboard/menu usage.
- Has a top 10 shards leaderboard through ajLeaderboards and DecentHolograms.

Custom Join / Leave / Vanish Messages
-------------------------------------
Custom join and leave messages should be server-wide, not staff-only.

Real join:
- Sends custom join message to all players.

Real leave:
- Sends custom leave message to all players.

/vanish on:
- Sends fake leave message to all players.

/vanish off:
- Sends fake join message to all players.

The custom script suppresses default join/quit where needed and sends the
CannaSMP messages itself.

Server Notice / Consent GUI
---------------------------
The server has a CannaSMP notice and consent system.

Players should be shown a GUI warning about the server's cannabis/mature
theme.

If they accept:
- They are allowed to play.
- They should not be asked again.
- Acceptance is stored in a file.

If they decline:
- They are kicked and asked to try again if they want to play.
- Decline count is stored.

Notice command aliases:
- /serverwarning
- /warning
- /notice

Create a public CannaSMP stats website hosted through GitHub Pages and later
connected to a custom domain.

Possible domain setup:
- cannasmp.net
- stats.cannasmp.net

Recommended architecture:
1. LiteByte Minecraft server exports stats.
2. A plugin writes stats to JSON files.
3. A scheduled process or plugin push sends updated JSON to GitHub.
4. GitHub Pages hosts the website.

Operational Notes
-----------------
Always back up live configs/jars before editing.

Use SFTP for file transfer.

Use the LiteByte panel console for:
- reload commands
- restarts
- checking errors

Many plugin command registrations require a full server restart after replacing
a jar. Skript changes can usually be reloaded with:
sk reload <script-name>

Avoid changing unrelated configs when making targeted fixes.

Useful Console Commands
-----------------------
Reload a Skript:
sk reload custom-join-leave

Reload all Skripts:
sk reload all

Reload TAB:
tab reload

Reload DecentHolograms:
dh reload

Reload ajLeaderboards data:
ajlb reload

Update ajLeaderboards offline board:
ajlb updatealloffline <board>

Restart server:
